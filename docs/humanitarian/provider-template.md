# Important

You **do not** need to investigate technical details.

Please **do not** spend time trying to determine:

- APIs
- JSON endpoints
- Authentication
- HTML scraping
- Programming technologies

Those tasks will be handled later by the development team.

Your contribution is to help GeoResponde discover valuable humanitarian data sources that may become future providers.

# Submission

Please submit your findings as either:

- A Markdown (`.md`) document
- A Text (`.txt`) document

Each provider should follow this template.

The document should contain enough information for developers to evaluate whether a new GeoResponde provider should be considered for implementation.
